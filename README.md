# CyclePath2Life - React Website

A modern, futuristic React website for CyclePath2Life, rebuilt with a cutting-edge design theme featuring neon gradients, glassmorphism, and smooth animations.

## Features

- **Modern React Architecture**: Built with React 18+ and Vite for fast development and optimal performance
- **Futuristic Design**: Dark tech theme with neon accents, glassmorphism cards, and smooth animations
- **Framer Motion Animations**: Engaging scroll animations and interactive elements
- **Fully Responsive**: Mobile-first design that works perfectly on all devices
- **SEO Optimized**: Complete meta tags and semantic HTML structure
- **Clean Component Structure**: Well-organized, reusable components

## Tech Stack

- **React 18+** - UI library
- **Vite** - Build tool and dev server
- **React Router DOM** - Client-side routing
- **Framer Motion** - Animation library
- **Lucide React** - Icon library
- **CSS** - Custom styling with CSS variables

## Getting Started

### Installation

```bash
cd cyclepath
npm install
npm run dev
```

Visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
